想给Framework7做一些贡献吗

你可以有几种方式类给Framework7的开发做一些贡献，让她变得越来越好：

在github上提交你的代码

如果你有任何可以改进 Framework7 的地方，欢迎在 Framework7 GitHub project 上提交你的代码。

如果你有任何建议或者问题，请在 submitting an issue 上提交。

提交你的案例

你可以把自己的应用提交到 Demo Apps 或者 Examples，这会对新的开发者有很大帮助。

捐助

如果你觉得Framework7很好，你可以 捐助作者。
